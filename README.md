Multimedia Admin System for WandouLabs
===========

install:

1  clone

2 `npm install`

3 `bower install`

4 run server: `grunt server`

http://0.0.0.0:9999/

------

if no grunt-cli:

`npm install -g grunt-cli`

------
if no grunt-contrib-compass

`npm install grunt-contrib-compass`

------

if no Compass:

`sudo gem update --system`

`sudo gem install compass`
